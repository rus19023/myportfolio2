Cassandra

Cassandra, from Apache, is a popular NoSQL database written in Java, originally made for Facebook, which has an enormous amount of data. It is a distributed, column-store database, with built-in support for multiple languages, that "has no single point of failure". It is designed to be highly available and highly horizontally scalable.

Designing a model with Cassandra is not like using a relational database. The design is based on what needs to be queried for the business' purposes. Other famous companies that use Cassandra are Ebay, Hulu, Twitter and Uber.

MapReduce

Cassandra works by spreading out data across multiple servers called nodes, and then replicating it to other nodes. This is called a "distributed system". This database system is best for businesses that cannot afford to have downtime or lost data.

MapReduce is the basis for Hadoop, which often runs with Cassandra. It is a way to process data in parallel, and then combine the results, using "worker nodes" that apply map functions to the data, shuffle the data between nodes using the output keys from the map functions, then reduce the results by processing them in parallel using the unique keys generated during the previous steps.

CQL

CQL is Cassandra's version of SQL. It is very limited compared to SQL, as it cannot do joins, subqueries, or even aggregate functions. It also has very similar syntax compared to SQL, therefore rather easy to learn for experienced database developers.
